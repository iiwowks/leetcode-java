# 动态规划模板

* [wiki定义](https://zh.wikipedia.org/wiki/%E5%8A%A8%E6%80%81%E8%A7%84%E5%88%92)
* 动态规划和**递归或者分治**没有根本上的区别（关键看有无最优子结构）
* 共性：找到重复子问题
* 差异性：**最优子结构**、**中途必须淘汰次优解**

## DP步骤

1. **重复子问题**（分治），最优子结构: opt\[n\] = best\_of\(opt\[n - 1\], opt\[n - 2\], ...\)
2. **递归和记忆化**或者**建立DP状态表**（自底向上递推），定义状态数组
3. **递推方程**、**DP方程**
